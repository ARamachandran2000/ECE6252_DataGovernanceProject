# Viewpoint B

This folder contains Viewpoint B code contributions and setup instructions.

## What is included

 - Setup_Dataset.ipynb: Setup script for downloading dataset
 - Viewpoint_B_Exp1.ipynb
 - Viewpoint_B_Exp2.ipynb

## Prerequisites and Setup

1) Make an account on Mozilla Data Collective to access the database: https://mozilladatacollective.com/
2) Generate a personal API Key: https://mozilladatacollective.com/api-reference
3) Replace TODO statements in Setup_Dataset.ipynb with the API key and GTID where indicated.
4) Run Setup_Dataset.ipynb inside Jupyter notebook in GT PACE cluster. 
5) Replace TODO statement in first cell of Viewpoint_B_Exp1.ipynb and Viewpoint_B_Exp2.ipynb with GTID
6) Now experiments 1 and 2 for viewpoint B can be run.
